# last messages.

A minimalist personal website where I write blog posts, notes, document projects, and occasionally build small puzzle games.

## Why I built it

I wanted a place on the internet that felt personal rather than another social media profile. Instead of constantly chasing new technologies, I wanted something simple that I could maintain for years.

## Features

* MDX-powered blog, notes, and projects
* Responsive design
* Interactive puzzle login
* Automatic sitemap and robots generation
* Dark minimalist theme

## Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* MDX

## Future Plans

* More puzzle games
* Better search
* Tags and archives
* Reading time estimation
* RSS feed
